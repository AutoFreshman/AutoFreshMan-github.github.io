城市拆法

三座城市的功能分工

昆明 落地适应 / 汇合 / 返程前夜

昆明 | 首尾节点

昆明段只承担 首晚适应、城市轻松散步 和 返程前夜 三个任务。

首晚和末晚都以接驳方便为主，主活动集中在 南屏街、昆明老街、翠湖 一带。

如果抵达时间早，可加一个 官渡古镇 或 斗南花市； 如果抵达时间晚，就安排晚餐和休息。

大理 自驾价值最明显的一站

大理 | 环海按两天组织

大理直接拆成 洱海西岸日、洱海东岸日 和 古城 / 苍山补点 三段。

洱海西岸重点是 生态廊道、崇圣寺三塔、喜洲古镇、海舌公园； 洱海东岸重点是 双廊、小普陀、文笔村、 理想邦 和 大理云想山。

丽江 雪山日和古镇日分开

丽江 | 雪山与古镇分天执行

丽江部分只分成两类：一类是 玉龙雪山 和 蓝月谷 的整天安排，一类是 白沙古镇、 束河古镇 这样的古镇补点。

雪山日不叠加古镇；回昆明当天只补一个北线片区，不再把全部慢游内容塞满。

逐日路书

04-17 晚到 04-25 晚的 8 晚 9 天执行表

04-17 夜间出发，04-18 昆明适应， 04-24 回昆明，04-25 夜间返程。

04-17 周五 武汉 2 人 / 西安 2 人夜间飞昆明，汇合入住

交通 两地分头飞昆明，按最晚到达的人收口。

玩法 不编正式景点，只完成汇合、入住、吃饭、休息。

住宿 南屏街 / 昆明老街 / 翠湖外圈 首晚。

04-18 周六 昆明适应日

交通 市内轻移动，不取车，不安排跨城。

玩法 按城市核心区走法执行：南屏街、昆明老街、翠湖、文明街 / 文化巷 四选三。

执行 这一天的目标是把 4 个人的作息和身体状态拉齐，不追景点数量。

04-19 周日 昆明补官渡 / 斗南，下午自驾去大理

交通 上午取车，补 官渡古镇 或 斗南花市，下午出发去大理。

玩法 抵达大理后只安排入住和 大理古城 夜游，不把环海内容塞进第一天。

执行 第一段长驾只完成进城与安顿，住宿不进古城深处。

04-20 周一 大理洱海西岸日

玩法 按视频拆法执行 生态廊道、崇圣寺三塔、喜洲古镇、海舌公园。

交通 全日自驾，但按片区停车后步行展开。

执行 上午水边，午间喜洲，下午海舌，晚上回古城外围休息。

04-21 周二 大理洱海东岸日

玩法 按 双廊古镇、小普陀、文笔村、理想邦、大理云想山日落 顺序组织。

交通 自驾，洱海东岸单独成天，不和洱海西岸混编。

执行 这一天控制停车次数，不做“见点就停”的低效率走法。

04-22 周三 大理补苍山半天，下午自驾去丽江

交通 上午仍在大理，下午顺行去丽江。

玩法 早段可在 苍山索道 / 感通片区 和 古城补点 二选一，傍晚入住丽江后补 束河 或 白沙。

住宿 束河 / 清溪 / 古城北侧可停车片区。

04-23 周四 丽江雪山日

玩法 仅安排 玉龙雪山、蓝月谷 主线；不把古镇群和雪山日混编。

交通 车只负责到集散点或官方停车区，山上以景区交通为主。

执行 不把 4680 作为唯一目标，按天气和票务把 牦牛坪 / 云杉坪 作为稳方案。

04-24 周五 丽江北线补点，午后回昆明

玩法 优先补 白沙古镇 或 束河古镇 一个北线片区。

交通 午前结束丽江内容，午后直接自驾回昆明。

住宿 机场 / 官渡 / 环城南路 片区，完成还车、餐饮、行李整理。

04-25 周六 昆明白天收口，夜间分组返程

交通 按后续锁定的航班，从昆明分别返武汉和西安。

玩法 白天补轻松城市内容，如 斗南、官渡 或直接休整。

安排 省内长距移动在 04-24 前完成。

住宿落点

住宿片区标准

昆明首晚

片区标准：南屏街 / 老街 / 翠湖外圈。 目标：完成四人汇合、落地安顿与首晚餐饮。

选址标准：二环内但不住滇池远侧。视频转写里明确把这一片定义为昆明最热闹、最适合首刷落脚的区域。

大理 3 晚

片区标准：大理古城西侧 / 苍山脚下可停车民宿群。 功能：服务洱海西岸出发、洱海东岸回收、古城晚餐、稳定停车。

停车标准：古城外围带院停车民宿 高于 古城核心步行区内民宿。 住宿样式不追洱海边清水房，优先执行效率。

丽江 2 晚

片区标准：不进入古城最深处，优先 束河古镇、 清溪 或 古城北侧可停车民宿群。

功能：兼顾 雪山方向出发 与 白沙 / 玉湖 / 束河短驳。

昆明末晚

片区标准：以次日晚间去机场与还车便利度为主，不回老城核心区。

收口片区：官渡古镇片区 / 巫家坝 / 机场方向。 核心指标：还车便利、机场通勤、四人分组返程。

自驾细化

方向、停车与执行顺序

片区停车、方向顺行、减少折返。

昆明首晚 / 末晚片区选择

首晚 固定在 昆明老街 / 南强街 / 翠湖外圈； 末晚 固定在 官渡古镇片区 / 巫家坝 / 机场方向。

首晚服务会合与适应，末晚服务还车与返程，不采用同一片区重复入住策略。

大理西岸日 | 停一次走一段

执行顺序：大理古城出发 → 生态廊道 / 龙龛 / 才村 → 崇圣寺三塔 → 喜洲古镇 → 海舌公园。 停车逻辑：片区停车后步行展开，不采用沿线见点即停模式。

日内结构：上午水边、中午喜洲古镇、下午海舌公园或返古城。

大理东岸日 | 东岸单独成天

东岸独立成天。执行顺序： 大理古城出发 → 双廊古镇 → 小普陀 → 文笔村 / 理想邦 → 大理云想山。

控制项：停车次数。 标准节奏：双廊古镇停一次、小普陀停一次、洱海东岸观景点停一次。

丽江雪山日 | 车只负责到集散点

车辆职责：酒店 → 集散区 / 官方停车场。 后续流程：景区交通接续上山。

关键控制项：时间窗、票务、天气。

丽江补点半天 | 一条北线串起来

执行顺序：住处出发 → 白沙古镇或束河古镇 → 午后回昆明。

车辆用途：半天短驳 与 午后返昆。

整条线的停车原则

原则：住宿可停车、片区停车后步行、热门点早到。 禁止项：车辆直接压入核心步行街区；沿途高频挪车。

适用片区：大理古城、丽江古城外围、白沙古镇 周边。
