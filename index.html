路线判断

本页仅保留昆大丽自驾线的执行判断。

执行口径如下：昆明落地开局，云南省内主移动切到自驾；首晚不安排落地即长途，末晚提前回收至昆明。

判断一：昆明承担首尾枢纽

从相关笔记封面和小红书常见写法看，首刷云南通常采用“先落地昆明，再展开经典线”的结构。 因此昆明在本手册中定义为会合、适应与返程节点。

判断二：自驾价值集中在大理与丽江

环洱海、喜洲、双廊、白沙古镇、 玉湖村 这些点位在小红书里经常被组织为同一天或半天路线， 对应的最优交通方式是片区自驾短驳。

判断三：长驾段控制为两段

本手册把长驾段收束成 昆明 → 大理 与 丽江 → 昆明。 中段 大理 → 丽江 作为顺行衔接段处理，不增加额外交通切换。

昆明 → 自驾取车后出发 大理 → 自驾顺行 丽江 → 自驾回昆明收口 昆明 → 次晨返程 武汉 / 宜昌

昆明 | 首尾节点

小红书里和昆明绑定最紧的仍然是 斗南花市、海埂大坝、 昆明老街、南强街。因此昆明段按轻城市开场与返程收口组织。

自驾版本的车辆获取时间设为第二天。首晚与末晚的住宿标准以接驳顺畅、停车稳定、夜间驾驶最少为主。

参考视频贴《第一次来云南该如何选择路线？》的封面是一张 昆明旅游地图，把 翠湖、南屏街老街、 官渡古镇、斗南、海埂大坝、 西山风景区、海洪湿地、捞鱼河湿地 串成了“先落地昆明，再向外展开”的结构，进一步强化了昆明作为首尾节点的角色。

大理 自驾价值最明显的一站

大理 | 环海按两天组织

小红书里大理最稳定的关键词是 环洱海、喜洲古镇、 龙龛码头、双廊、大理古城。对应的执行方式是将西岸与东岸分天编排。

大理段固定为 3 晚：进城日、西岸日、东岸或留白日。车辆用于片区串联，不用于压缩天数。

小红书搜索 大理自驾 环洱海 喜洲古镇
丽江 雪山日和古镇日分开

丽江 | 雪山与古镇分天执行

小红书里丽江内容分成两组：玉龙雪山、蓝月谷 与 白沙古镇、玉湖村、束河。执行方式对应为强景观日与慢游日分开。

丽江段的车辆主要服务古镇群之间的移动。雪山日按景区交通优先、车辆停放在住处或集散点的口径执行。

逐日路书

04-18 到 04-26 的 8 晚 9 天执行表

编排标准：首晚不疲劳驾驶，末晚提前回昆明，城市内部按片区集中组织。

04-18 周六 武汉 / 宜昌飞昆明，市区轻落地

交通 飞昆明，首日不安排提车后直接长途。

玩法 会合、入住、餐饮；到达时间充裕时补 昆明老街 或 海埂大坝。

住宿 市区首晚。

04-19 周日 昆明取车，自驾去大理

交通 上午取车后出发，下午进入大理。

玩法 仅安排 大理古城 与夜间餐饮，不编入环海段。

执行 第一个长驾日仅完成进城与安顿。

04-20 周一 大理西岸日

玩法 按 龙龛、才村、喜洲 的西岸顺序执行。

交通 全日自驾。

执行 当日仅完成西岸段。

04-21 周二 大理东岸日或留白日

玩法 按 双廊、小普陀、东岸休憩点组织。

交通 自驾。

执行 根据天气与停车情况决定半环线或东岸完整日。

04-22 周三 自驾去丽江，傍晚白沙

交通 大理至丽江顺行。

玩法 到达后先入住，傍晚进入 白沙古镇 或周边日落点。

住宿 白沙 / 清溪 / 古城北侧 片区。

04-23 周四 丽江雪山日

玩法 仅安排 玉龙雪山、蓝月谷 主线。

交通 景区交通或包车优先。

执行 雪山日不叠加古镇群。

04-24 周五 丽江慢游日

玩法 自驾串联 白沙古镇、玉湖村、束河 或 忠义市场。

交通 自驾短驳。

执行 当日保留留白时段。

04-25 周六 自驾回昆明，住返程落点

交通 丽江回昆明是整条线第二个真正长驾日，尽量早出发。

玩法 不编正式景点，只完成还车、餐饮、行李整理。

住宿 机场 / 官渡 / 环城南路 片区。

04-26 周日 昆明返程

交通 按后续锁定的航班，从昆明返武汉 / 宜昌。

逻辑 上一晚必须已经收回昆明，避免把返程晨间风险留给高速。

住宿落点

住宿片区标准

昆明首晚

片区标准：老街 / 南强街 / 翠湖外圈。 目标：完成落地安顿与城市首晚餐饮。

选址标准：二环内但不进入最窄巷子，便于次日取车与出城。

大理 3 晚

片区标准：古城外沿 或 古城与洱海西岸之间。 功能：服务西岸出发、古城晚餐、稳定停车。

停车标准：古城外围带院停车民宿 高于 古城核心步行区内民宿。

丽江 3 晚

片区标准：不进入古城最深处，优先 白沙、 清溪 或 古城北侧可停车民宿群。

功能：兼顾 雪山方向出发 与 白沙 / 束河短驳。

昆明末晚

片区标准：以次晨去机场与还车便利度为主，不回老城核心区。

收口片区：官渡 / 巫家坝 / 机场方向。 核心指标：次晨通勤。

自驾细化

方向、停车与执行顺序

本节只定义驾驶顺序与停车标准，不扩展景点数量。执行原则为片区停车、方向顺行、减少折返。

昆明首晚 / 末晚片区选择

首晚 固定在 老街 / 南强街 / 翠湖外圈； 末晚 固定在 官渡 / 巫家坝 / 机场方向。

首晚服务适应，末晚服务返程，不采用同一片区重复入住策略。

大理西岸日 | 停一次走一段

执行顺序：古城出发 → 龙龛 / 才村 → 喜洲。 停车逻辑：片区停车后步行展开，不采用沿线见点即停模式。

日内结构：上午水边、中午喜洲、下午返古城。

大理东岸日 | 东岸单独成天

东岸独立成天。执行顺序： 古城出发 → 双廊 → 小普陀 → 视天气决定是否继续南下。

控制项：停车次数。 标准节奏：双廊停一次、小普陀停一次。

丽江雪山日 | 车只负责到集散点

车辆职责：酒店 → 集散区 / 官方停车场。 后续流程：景区交通接续上山。

关键控制项：时间窗、票务、天气。

白沙慢游日 | 一条北线串起来

执行顺序：住处出发 → 白沙古镇 → 玉湖村 → 返程补束河或忠义市场。 白沙与玉湖并入同一天，雪山日单独编排。

车辆用途：村与村之间切换 与 午后回城补点。

整条线的停车原则

原则：住宿可停车、片区停车后步行、热门点早到。 禁止项：车辆直接压入核心步行街区；沿途高频挪车。

适用片区：大理古城、丽江古城外围、白沙古镇周边。
