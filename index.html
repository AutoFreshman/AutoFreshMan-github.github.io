路线判断

本页仅保留昆大丽自驾线的执行判断。

判断一：昆明承担首尾缓冲

视频转写明确给出理由：昆明机票便宜、游客集散能力强、 比高海拔机场更适合首刷。因此昆明在本手册中不是主玩法城市，而是 会合、适应、返程收口 三合一节点。

判断二：自驾价值集中在大理与丽江

转写里最有价值的不是城际交通，而是拆天方法： 大理海西 / 海东分天、丽江雪山日单列、 白沙 / 玉湖 / 束河一线串。这些内容切到自驾后反而更顺。

判断三：不把夜路风险留给山线

视频后半段专门提醒了山路夜驾和高海拔日程的风险。当前版本对应的改法是： 04-17 只完成到昆明，04-24 提前把车和人都收回昆明， 04-25 只保留返程动作。

逐日路书

04-17 晚到 04-25 晚的 8 晚 9 天执行表

编排标准：04-17 只完成跨城会合，04-18 做昆明适应， 04-24 提前回昆明收口，04-25 只留返程。

04-17 周五 武汉 2 人 / 西安 2 人夜间飞昆明，会合入住

交通 两地分头飞昆明，按最晚到达的人收口。

玩法 不编正式景点，只完成会合、入住、吃饭、休息。

住宿 南屏街 / 老街 / 翠湖外圈 首晚。

04-18 周六 昆明适应日

交通 市内轻移动，不取车，不安排跨城。

玩法 按视频里最稳的城市核心区走法执行：南屏街、老街、翠湖、文明街 / 文化巷 四选三。

执行 这一天的目标是把 4 个人的作息和身体状态拉齐，不追景点数量。

04-19 周日 昆明补官渡 / 斗南，下午自驾去大理

交通 上午取车，补 官渡古镇 或 斗南花市，下午出发去大理。

玩法 抵达大理后只安排入住和 大理古城 夜游，不把环海内容塞进第一天。

执行 第一段长驾只完成进城与安顿，住宿不进古城深处。

04-20 周一 大理海西日

玩法 按视频拆法执行 生态廊道、崇圣寺三塔、喜洲古镇、海舌公园。

交通 全日自驾，但按片区停车后步行展开。

执行 上午水边，午间喜洲，下午海舌，晚上回古城外围休息。

04-21 周二 大理海东日

玩法 按 双廊古镇、小普陀、文笔村、理想邦、云想山日落 顺序组织。

交通 自驾，东岸单独成天，不和海西混编。

执行 这一天控制停车次数，不做“见点就停”的低效率走法。

04-22 周三 大理补苍山半天，下午自驾去丽江

交通 上午仍在大理，下午顺行去丽江。

玩法 早段可在 苍山索道 / 感通片区 和 古城补点 二选一，傍晚入住丽江后补 束河 或 白沙。

住宿 束河 / 清溪 / 古城北侧可停车片区。

04-23 周四 丽江雪山日

玩法 仅安排 玉龙雪山、蓝月谷 主线；不把古镇群和雪山日混编。

交通 车只负责到集散点或官方停车区，山上以景区交通为主。

执行 不把 4680 作为唯一目标，按天气和票务把 牦牛坪 / 云杉坪 作为稳方案。

04-24 周五 丽江北线补点，午后回昆明

玩法 只补一个北线片区，优先 白沙古镇 或 束河古镇，不再把 玉湖 和整条慢游线全部塞进来。

交通 午前结束丽江内容，午后直接自驾回昆明。

住宿 机场 / 官渡 / 环城南路 片区，完成还车、餐饮、行李整理。

04-25 周六 昆明白天收口，夜间分组返程

交通 按后续锁定的航班，从昆明分别返武汉和西安。

玩法 白天只补轻松城市内容，如 斗南、官渡 或直接休整，不再安排高速。

逻辑 返程日只保留城市内机动，所有省内长距移动必须在 04-24 前完成。

住宿落点

住宿片区标准

昆明首晚

片区标准：南屏街 / 老街 / 翠湖外圈。 目标：完成四人会合、落地安顿与首晚餐饮。

选址标准：二环内但不住滇池远侧。视频转写里明确把这一片定义为昆明最热闹、最适合首刷落脚的区域。

大理 3 晚

片区标准：古城西侧 / 苍山脚下可停车民宿群。 功能：服务海西出发、海东回收、古城晚餐、稳定停车。

停车标准：古城外围带院停车民宿 高于 古城核心步行区内民宿。 住宿样式不追洱海边清水房，优先执行效率。

丽江 2 晚

片区标准：不进入古城最深处，优先 束河、 清溪 或 古城北侧可停车民宿群。

功能：兼顾 雪山方向出发 与 白沙 / 玉湖 / 束河短驳。

昆明末晚

片区标准：以次日晚间去机场与还车便利度为主，不回老城核心区。

收口片区：官渡 / 巫家坝 / 机场方向。 核心指标：还车便利、机场通勤、四人分组返程。

自驾细化

方向、停车与执行顺序

本节只定义驾驶顺序与停车标准，不扩展景点数量。执行原则为片区停车、方向顺行、减少折返。

昆明首晚 / 末晚片区选择

首晚 固定在 老街 / 南强街 / 翠湖外圈； 末晚 固定在 官渡 / 巫家坝 / 机场方向。

首晚服务会合与适应，末晚服务还车与返程，不采用同一片区重复入住策略。

大理西岸日 | 停一次走一段

执行顺序：古城出发 → 生态廊道 / 龙龛 / 才村 → 三塔 → 喜洲 → 海舌。 停车逻辑：片区停车后步行展开，不采用沿线见点即停模式。

日内结构：上午水边、中午喜洲、下午海舌或返古城。

大理东岸日 | 东岸单独成天

东岸独立成天。执行顺序： 古城出发 → 双廊 → 小普陀 → 文笔村 / 理想邦 → 云想山。

控制项：停车次数。 标准节奏：双廊停一次、小普陀停一次、海东视觉点停一次。

丽江雪山日 | 车只负责到集散点

车辆职责：酒店 → 集散区 / 官方停车场。 后续流程：景区交通接续上山。

关键控制项：时间窗、票务、天气。

白沙慢游日 | 一条北线串起来

执行顺序：住处出发 → 白沙古镇 → 玉湖村 → 束河古镇 → 丽江古城夜间补点。 白沙与玉湖并入同一天，雪山日单独编排。

车辆用途：村与村之间切换 与 午后回城补点。这是视频里最清晰的一条不绕路慢游线。

整条线的停车原则

原则：住宿可停车、片区停车后步行、热门点早到。 禁止项：车辆直接压入核心步行街区；沿途高频挪车。

适用片区：大理古城、丽江古城外围、白沙古镇周边。
