住宿落点

住宿片区标准

昆明首晚

片区标准：南屏街 / 老街 / 翠湖外圈。 目标：完成集合、落地安顿与首晚餐饮。

选址标准：二环内、靠近老街与翠湖片区。视频转写里明确把这一片定义为昆明最热闹、最适合首刷落脚的区域。

大理 3 晚

片区标准：大理古城西侧 / 苍山脚下可停车民宿群。 功能：服务洱海西岸出发、洱海东岸回收、古城晚餐、稳定停车。

停车标准：古城外围带院停车民宿 高于 古城核心步行区内民宿。 住宿样式不追洱海边清水房，优先执行效率。

丽江 2 晚

片区标准：优先 束河古镇、 清溪 或 古城北侧可停车民宿群。

功能：兼顾 雪山方向出发 与 白沙古镇 / 玉湖村 / 束河古镇 之间的短距离移动。

昆明末晚

片区标准：以次日晚间去机场与还车便利度为主。

返程前夜片区：官渡古镇片区 / 巫家坝 / 机场方向。 核心指标：还车便利、机场通勤、次日早班机出发效率。

自驾细化

方向、停车与执行顺序

片区停车、方向顺行、减少折返。

昆明首晚 / 末晚片区选择

首晚 固定在 昆明老街 / 南强街 / 翠湖外圈； 末晚 固定在 官渡古镇片区 / 巫家坝 / 机场方向。

首晚服务集合与适应，末晚服务还车与返程。

大理西岸日 | 停一次走一段

执行顺序：大理古城出发 → 生态廊道 / 龙龛 / 才村 → 崇圣寺三塔 → 喜洲古镇 → 海舌公园。 停车逻辑：片区停车后步行展开。

日内结构：上午水边、中午喜洲古镇、下午海舌公园或返古城。

大理东岸日 | 东岸单独成天

东岸独立成天。执行顺序： 大理古城出发 → 双廊古镇 → 小普陀 → 文笔村 / 理想邦 → 大理云想山。

停车节奏：双廊古镇停一次、小普陀停一次、洱海东岸观景点停一次。

丽江雪山景区日 | 外围轻松游

车辆职责：酒店 → 集散区 / 官方停车场。 后续流程：景区接驳车 + 步行完成外围游览。

关键关注点：天气、景区开放、回城时间。

丽江半天安排 | 一条北线串起来

执行顺序：住处出发 → 白沙古镇或束河古镇 → 午后回昆明。

车辆用途：半天短距离移动 与 午后返回昆明。

整条线的停车原则

原则：住宿可停车、片区停车后步行、热门点早到。 禁止项：车辆直接压入核心步行街区；沿途高频挪车。

适用片区：大理古城、丽江古城外围、白沙古镇 周边。
