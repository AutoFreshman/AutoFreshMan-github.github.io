# 2026 昆大丽自驾行程规划

适用时间窗：`2026-04-17` 晚间出发，`2026-04-18 00:00` 左右抵昆明，`2026-04-26` 从昆明返程  
出发城市：`武汉 / 宜昌 / 西安`  
路线固定：`昆明 -> 大理 -> 丽江 -> 昆明`  
执行原则：云南省内以 `自驾` 为主，但不把疲劳驾驶当成自由度

## 规划前提

- 文档仅保留一条路线，不比较其他方案。
- 页面和文档的路线判断只参考 `小红书`。
- 参考起点是小红书笔记《`第一次来云南该如何选择路线？`》，其封面明确出现了 `七天时间云南怎么玩` 和 `推荐大家落地昆明` 的判断。
- 结合 `2026-04-10` 在已登录 Chrome 小红书里的检索结果，昆明段改为优先保留 `斗南花市`、`海埂 / 滇池`、`西山`、`捞鱼河湿地公园` 这一组；`翠湖` 不再作为主景点，`讲武堂` 因可能维修不再写成刚性主线。
- 因为末端返程仍在 `昆明`，所以最后一晚必须已经回到昆明，不把返程风险留给高速。

## 自驾适配依据

1. `昆明 -> 大理 -> 丽江 -> 昆明` 本身就是首刷云南最成熟的一条经典线，路线不会拧巴。
2. 自驾段主要配置给 `大理环洱海`、`白沙古镇 / 玉湖村` 等内部移动密集区域。
3. `大理 3 晚`、`丽江 2 晚` 的配置用于拆分环海日、雪山日与返程前回收。

## 城市分工

### 昆明

- 负责 `凌晨会合入住`、`完整城市日`、`返程前收口`。
- 昆明段主线改成 `斗南花市` 和 `滇池 / 海埂 / 西山` 两组，`捞鱼河湿地公园` 作为天气和体力都合适时的自然向备选；`翠湖`、`讲武堂` 不再作为核心安排。
- 若花期对上，可把 `教场中路蓝花楹` 作为四月加分项，而不是刚性主线。
- `04-17` 只负责飞行与抵达，昆明完整城市内容从 `04-18` 白天开始。

### 大理

- 负责整条线里最明显的 `自驾收益`。
- 小红书高频词集中在 `环洱海`、`喜洲古镇`、`龙龛码头`、`双廊`、`大理古城`。
- 执行标准：`西岸` 与 `东岸` 分天编排。

### 丽江

- 负责 `雪山强景观` 和 `古镇慢游` 两种节奏。
- 小红书高频词集中在 `玉龙雪山`、`白沙古镇`、`玉湖村`、`束河`。
- 车辆用途：古镇群与村落短驳；雪山日不要求把车辆直接开到最深处。

## 推荐住法

- `昆明 2 晚 + 大理 3 晚 + 丽江 2 晚 + 昆明 2 晚`

执行标准：

- 昆明前两晚：市区片区。
- 昆明后两晚：返程与还车便利片区。
- 大理：`古城外沿` 或 `古城与西岸之间`。
- 丽江：`白沙 / 清溪 / 古城北侧可停车区域`。

### 昆明前两晚 / 后两晚的具体落脚片区

- `前两晚` 片区：`翠湖外圈 / 南屏街 / 南强街`。
- `后两晚` 片区：`巫家坝 / 机场方向`。
- 片区策略：前段解决 `4月18日0点左右` 的落地入住与次日步行游逛，末段解决还车与返程效率。

## 自驾执行版

### 昆明城市日 | 04-18 执行顺序

- 推荐顺序：`斗南花市 -> 海埂大坝 -> 西山 -> 南强街`。
- 上午：先去 `斗南花市`，按 `买花 / 拍照 / 快速逛馆` 的节奏展开，不把这段拖到夜间。
- 下午前段：转去 `海埂大坝` 看滇池，主打 `湖边步行 + 视当天情况看海鸥`。
- 下午后段：接 `西山` 或 `滇池索道`，把昆明这一天的主景观收在高处视角。
- 晚上：回 `南强街 / 昆明老街` 吃饭收口，不再补 `翠湖`、`讲武堂` 这类老城区点。

### 昆明返程前一日 | 04-25 城市特色线

- 推荐顺序：`昆明老街 -> 黄公东街 -> 巡津街 -> 南强街`。
- 核心目标：把 `老街街巷`、`老城立面`、`咖啡与小店密度`、`夜间烟火气` 串成一条轻松但有城市辨识度的路线。
- 前半段：从 `昆明老街` 起步，顺着 `黄公东街` 一带慢走，重点看老城区肌理和街边店铺。
- 后半段：接 `巡津街`，这段更偏 `梧桐 / 洋楼 / 咖啡街区` 的城市气质。
- 晚上：回 `南强街` 吃饭收口，同时完成 `还车`、`行李整理`、`次日值机准备`。

### 大理环洱海 | 西岸日

- 执行顺序：`古城出发 -> 龙龛 / 才村 -> 喜洲 -> 返古城`。
- 停车逻辑：`片区停车后步行展开`。
- 日内结构：`上午水边 + 中午喜洲 + 下午返古城`。

### 大理环洱海 | 东岸日

- 执行顺序：`古城出发 -> 双廊 -> 小普陀 -> 视天气决定是否继续南下`。
- 控制项：减少碎停车与反复折返。
- 标准节奏：`双廊停一次`、`小普陀停一次`。

### 丽江雪山日

- 车辆职责：`酒店 -> 集散区 / 官方停车场`。
- 后续流程：景区交通接续上山。
- 编排标准：雪山日不叠加白沙、束河等古镇内容。

### 丽江白沙慢游日

- 执行顺序：`住处出发 -> 白沙古镇 -> 玉湖村 -> 回城补束河或忠义市场`。
- 编排标准：白沙与玉湖并入同一天，雪山日单独安排。
- 车辆用途：`村与村之间切换` 与 `午后回城补点`。

### 整体停车原则

1. 落脚点标准：`住宿可停车`。
2. 热门片区标准：`停一次走一段`。
3. 禁止项：车辆压入大理古城、丽江古城、白沙古镇等片区的最核心步行区域。

## 执行重点

1. `04-17` 晚间飞行，`04-18 0点左右` 到昆明后直接入住，`04-18` 白天再展开完整昆明城市内容，返程前夜继续提前回昆明。
2. 大理执行口径：`西岸` 与 `东岸` 分别成天。
3. 丽江执行口径：`雪山日` 与 `慢游日` 分开。
4. 雪山日采用景区交通优先口径。
